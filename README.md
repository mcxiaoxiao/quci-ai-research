# 中国传统曲艺生成 AI 研报包

这是一个围绕“曲词/曲艺生成 AI”做的市场与技术可行性研究仓库。

目标不是做一篇泛泛而谈的 AI 展望，而是把问题拆开：

- 这个方向到底有没有市场
- 技术上哪一层可行，哪一层大概率失败
- 数据、版权、评估、产品化的硬门槛在哪里
- 如果要做，应该从什么产品形态开始

仓库内容会包含：

- `report/chapters/`：分章节 Markdown 正文
- `report/appendix/`：证据库与来源索引
- `data/`：结构化来源数据
- `scripts/`：抓取证据、合并 Markdown、导出 PDF 的脚本
- `build/`：最终 HTML / PDF 输出

构建方式会采用 Markdown + HTML + PDF 的组合，正文按多个 Markdown 文件维护，最终再组装成专业版 PDF。
